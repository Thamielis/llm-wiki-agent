## Summary
Compact checklist of foundational cmdlets for administrators, spanning discovery (`Get-Help`, `Get-Command`, `Get-Member`), output handling (`Out-File`, `Export-Csv`, `Out-GridView`), execution (`Invoke-Item`), and control-flow primitives (`foreach`, `if`).

## Key Claims
- Core cmdlet fluency is a baseline requirement for PowerShell operations work.
- Object inspection and output shaping are as important as command execution itself.
- Looping/branching constructs are necessary progression steps beyond single-command usage.
