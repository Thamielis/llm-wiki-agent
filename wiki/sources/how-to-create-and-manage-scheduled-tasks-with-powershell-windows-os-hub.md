## Summary
Hands-on reference for creating, modifying, querying, exporting, and importing scheduled tasks using the `ScheduledTasks` module and fallback compatibility patterns (`schtasks`, COM) for older environments.

## Key Claims
- `ScheduledTasks` cmdlets provide full-lifecycle management of scheduler jobs.
- Trigger/action/principal composition enables reliable unattended task execution.
- XML export/import enables task portability across hosts.
